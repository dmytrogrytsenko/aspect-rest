Aspect REST API Documentation

PUT /targets/:id

Update the target.

Request

PUT http://your-it.com.ua:8888/targets/{targetId}

Request header

Name Description Example
Authorization The token to authorize the user. Authorization: 15d07e2d-6e0d-470f-8de9-3a8ce0afa934

Request parameters

Name Type Necessity Description Example
targetId String Required The unique identifier of the target. 3450ea8d-a532-474a-8974-55a0b4d519f7

Request body

Name Type Necessity Description Example
name String Optional The target name. "name": "Lancer"
keywords String Optional The target keywords. "keywords": "mitsubishi lancer"

Example of request

PUT http://your-it.com.ua:8888/targets/3450ea8d-a532-474a-8974-55a0b4d519f7
Authorization: 15d07e2d-6e0d-470f-8de9-3a8ce0afa934
{
    "name": "Lancer"
    "keywords": "mitsubishi lancer"
}

Response

Status Code Description
200 OK   The target is updated successfully.
401 Unauthorized The resource requires authentication, which was not supplied with the request.
The supplied authentication is invalid.
403 Forbidden The supplied authentication is not authorized to access this resource.
404 Not Found Project is not found.
Target is not found.
