Aspect REST API Documentation

GET /targets

Return the project targets.

Request

GET http://your-it.com.ua:8888/targets?projectId={projectId}

Request header

Name Description Example
Authorization The token to authorize the user. Authorization: 15d07e2d-6e0d-470f-8de9-3a8ce0afa934

Request parameters

Name Type Necessity Description Example
projectId String Required The unique identifier of the project. 0c1b0827-c10a-4fe2-849f-40aba9108032

Example of request

GET http://your-it.com.ua:8888/targtes?projectId=0c1b0827-c10a-4fe2-849f-40aba9108032
Authorization: 15d07e2d-6e0d-470f-8de9-3a8ce0afa934

Response

Status Code Description
200 OK   The project targets are returned successfully.
401 Unauthorized The resource requires authentication, which was not supplied with the request.
The supplied authentication is invalid.
403 Forbidden The supplied authentication is not authorized to access this resource.
404 Not Found Project is not found.

See Error Codes & Responses

Response body

Name Type Necessity Description Example
targets List of targets Required The list of project targets. "targets": [ {...}, {...}, ... ]

Target format

Name Type Necessity Description Example
id String Required The unique identifier of the target. "id": "3450ea8d-a532-474a-8974-55a0b4d519f7"
name String Required The name of the target. "name": "Mitsubishi"
keywords String Required The keywords of the target. "keywords": "mitsubishi lancer outlander pajero l200 colt"
