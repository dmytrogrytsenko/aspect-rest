Aspect REST API Documentation

GET /profile

Return the user profile information.

Request

GET http://your-it.com.ua:8888/profile

Request header

Name Description Example
Authorization The token to authorize the user. Authorization: 15d07e2d-6e0d-470f-8de9-3a8ce0afa934

Example of request

GET http://your-it.com.ua:8888/profile
Authorization: 15d07e2d-6e0d-470f-8de9-3a8ce0afa934

Response

Status Code Description
200 OK   The user profile is returned successfully.
401 Unauthorized The resource requires authentication, which was not supplied with the request.
The supplied authentication is invalid.

See Error Codes & Responses

Response body

Name Type Necessity Description Example
userId String Required The unique identifier of the user. "id": "98b1def7-e6dd-44e5-a369-54144f5a4232"
username String Required The name of the user. "name": "janedoe"
email String Required The email of the user. "email": "jane.doe@gmail.com"
firstName String Optional The first name of the user. "firstName": "Jane"
lastName String Optional The last name of the user. "lastName": "Doe"
company String Optional The company of the user. "company": "Good Corporation"
