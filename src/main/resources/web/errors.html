Error Codes & Responses

Response

Status Code Description
200 OK   The operation is performed successfully.
400 Bad Request The request's encoding is corrupt.
The form field was malformed.
The value of HTTP header was malformed.
The query parameter was malformed.
The request content was malformed.
Request is missing required cookie.
Request is missing required form field.
Request is missing required HTTP header.
Request is missing required query parameter.
Request entity expected but not supplied.
Uri scheme not allowed.
The request content validation is failed.
The request's Content-Encoding is not supported.
401 Unauthorized The resource requires authentication, which was not supplied with the request.
The supplied authentication is invalid.
403 Forbidden The supplied authentication is not authorized to access this resource.
404 Not Found Project is not found.
The requested resource could not be found.
Target is not found.
405 Method Not Allowed HTTP method not allowed.
406 Not Acceptable Resource representation is only available with Content-Types.
Resource representation is only available with Content-Encodings.
415 Unsupported Media Type There was a problem with the requests Content-Type.
416 Requested Range Not Satisfiable Request contains too many ranges.
The requested Ranges are not satisfiable.
500 Internal Server Error There was an internal server error.
The request processing is timed out.
