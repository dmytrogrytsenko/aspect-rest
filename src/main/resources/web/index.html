Aspect REST API Documentation

Error Codes & Responses

User operations

Method Resource Description
POST /login Login the user.
POST /logout Logout the user.
GET /profile Return the user profile information.

Project operations

Method Resource Description
GET /projects Return the user projects.
GET /projects/:id Return the project information.
POST /projects Add new project.
DELETE /projects/:id Remove the project.
PUT /projects/:id Update the project.

Target operations

Method Resource Description
GET /targets Return the project targets.
GET /targets/:id Return the target information.
POST /targets Add new target.
DELETE /targets/:id Remove the target.
PUT /targets/:id Update the target.
