Aspect REST API Documentation

POST /targets

Add new target.

Request

POST http://your-it.com.ua:8888/targets

Request header

Name Description Example
Authorization The token to authorize the user. Authorization: 15d07e2d-6e0d-470f-8de9-3a8ce0afa934

Request body

Name Type Necessity Description Example
projectId String Required The unique project identifier. "projectId": "0c1b0827-c10a-4fe2-849f-40aba9108032"
name String Required The target name. "name": "Mitsubishi"
keywords String Required The target keywords. "keywords": "mitsubishi lancer outlander pajero l200 colt"

Example of request

POST http://your-it.com.ua:8888/targets
Authorization: 15d07e2d-6e0d-470f-8de9-3a8ce0afa934
{
    "projectId": "0c1b0827-c10a-4fe2-849f-40aba9108032",
    "name": "Mitsubishi",
    "keywords": "mitsubishi lancer outlander pajero l200 colt"
}

Response

Status Code Description
200 OK   The target is added successfully.
400 Bad Request

The request content validation is failed:

  • Required member 'name' is empty.
    •
    401 Unauthorized The resource requires authentication, which was not supplied with the request.
    The supplied authentication is invalid.
    403 Forbidden The supplied authentication is not authorized to access this resource.
    404 Not Found Project is not found.

    See Error Codes & Responses
